# Bangla Chatbot 🤖

Bangla Chatbot is a Streamlit-based AI chatbot application that answers user questions entirely in Bangla. It supports English, Banglish, and Bangla input while always responding in pure Bangla. The project uses LangChain, OpenAI GPT models, and chat memory to create natural conversational interactions with session-based history support. Users can also choose different GPT models and control response length through the sidebar settings.
